# Relmdditallery

Simple reddit gallery written in elm

## Installation

### Development
```sh
$ npm install
$ npm run dev
```
Open localhost:3000 and got for it!

### Production
```sh
$ npm install
$ npm run prod
```
You find everything you need in dist/
